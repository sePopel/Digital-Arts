系统要求
------------------------------------------

Windows系统：
*Windows® 10, 11 或更高版本

Macintosh系统: 
*macOS® 10.15, 10.16 或更高版本

Windows和Macintosh的通用要求：
*需要64位操作系统
*2.0 GHz Intel Core i3或更高版本
*4 GB RAM或更高
*1.6 GB或更大的可用硬盘空间
*需要支持OpenGL 4.1的图形卡（建议使用1920x1080）**
*需要连接互联网激活产品

**对于高像素密度显示器，最低分辨率将根据操作系统建议的缩放级别而变化. 例如，如果操作系统建议将缩放标准设置为200％，则最低要求可能高达2736x1824分辨率. 或者，如果操作系统建议缩放标准设置为150％，则最低要求可能高达2160x1440分辨率.
