------------------------------------------
sistem gereksinimleri
------------------------------------------

Pencereler:
*Windows® 10, 11 veya üstü

Macintosh:
*macOS® 10.15, 10.16 veya üstü

Windows ve Macintosh için Ortak Gereksinimler:
*64-bit işletim sistemi gereklidir
*2.0 GHz Intel Core i3 veya üstü
*4 GB RAM veya üzeri
*1,6 GB boş sabit disk alanı veya üzeri
*OpenGL 4.1 destekli grafik kartı gereklidir (1920x1080 önerilir)**
*Ürün aktivasyonu için internet bağlantısı gereklidir

**Yüksek piksel yoğunluklu ekranlar için minimum çözünürlük, işletim sistemi tarafından önerilen ölçeklendirme düzeyine göre değişir. Örneğin, işletim sistemi %200'lük bir ölçekleme düzeyi önerirse, minimum gereksinim 2736x1824 çözünürlüğe kadar çıkabilir. Veya işletim sistemi %150'lik bir ölçekleme düzeyi önerirse, minimum gereksinim 2160x1440 çözünürlüğe kadar çıkabilir.
