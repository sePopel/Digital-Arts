잃어버린 대리석 약속
------------------------------------------
고객에 대한 약속을 이행하기 위한 노력의 일환으로 당사 제품에 대한 업데이터를 지속적으로 게시합니다. 업데이트를 확인하려면 www.lostmarble.com/updates를 방문하십시오.

Moho® 14 등록
------------------------------------------

Moho® 14 제품 등록은 설치 프로세스 중에 이루어집니다.


------------------------------------------
콘텐츠 배포
------------------------------------------

기존 Moho 콘텐츠 수정 및 배포에 대한 자세한 내용은 이 소프트웨어에 포함된 Moho - 최종 사용자 라이선스 계약("EULA")의 섹션 5("콘텐츠 라이선스")를 참조하십시오.


------------------------------------------
시스템 요구 사항
------------------------------------------

윈도우:
*Windows® 10, 11 이상

매킨토시:
*macOS® 10.15, 10.16 이상

Windows 및 Macintosh용 일반 요구 사항:
*64비트 OS 필요
*2.0GHz Intel Core i3 이상
*4GB RAM 이상
*1.6GB 이상의 여유 하드 드라이브 공간
*OpenGL 4.1 지원 그래픽 카드 필요(1920x1080 권장)**
*제품 활성화를 위한 인터넷 연결 필요

**높은 픽셀 밀도 디스플레이의 경우 최소 해상도는 운영 체제 권장 스케일링 수준에 따라 달라집니다. 예를 들어 OS가 200%의 확장 수준을 권장하는 경우 최소 요구 사항은 2736x1824 해상도만큼 높을 수 있습니다. 또는 OS에서 배율 수준 150%를 권장하는 경우 최소 요구 사항은 2160x1440 해상도만큼 높을 수 있습니다.
