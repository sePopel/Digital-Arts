Requisits del sistema
-------------------------------------------

Windows:
*Windows® 10, 11 o superior

Macintosh:
*macOS® 10.15, 10.16 o superior

Requisits comuns per a Windows i Macintosh:
*Cal un sistema operatiu de 64 bits
*2,0 GHz Intel Core i3 o superior
*4 GB de RAM o superior
*1,6 GB d'espai lliure al disc dur o superior
*Cal una targeta gràfica compatible amb OpenGL 4.1 (es recomana 1920 x 1080)**
*Cal connexió a Internet per activar el producte

**Per a pantalles d'alta densitat de píxels, la resolució mínima variarà segons el nivell d'escala recomanat del sistema operatiu. Per exemple, si el sistema operatiu recomana un nivell d'escala del 200%, el requisit mínim pot arribar a una resolució de 2736 x 1824. O si el sistema operatiu recomana un nivell d'escala del 150%, el requisit mínim pot arribar a una resolució de 2160 x 1440.
