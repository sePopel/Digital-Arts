Requisitos de sistema
------------------------------------------

Janelas:
*Windows® 10, 11 ou superior

Macintosh:
*macOS® 10.15, 10.16 ou superior

Requisitos comuns para Windows e Macintosh:
*SO de 64 bits necessário
*Intel Core i3 de 2,0 GHz ou superior
*4 GB de RAM ou superior
*1,6 GB de espaço livre no disco rígido ou superior
*Necessária placa de vídeo compatível com OpenGL 4.1 (recomenda-se 1920x1080)**
*Conexão com a Internet necessária para ativação do produto

**Para exibições de alta densidade de pixels, a resolução mínima varia de acordo com o nível de dimensionamento recomendado pelo sistema operacional. Por exemplo, se o sistema operacional recomendar um nível de escala de 200%, o requisito mínimo pode ser de até 2736 x 1824 de resolução. Ou se o sistema operacional recomendar um nível de escala de 150%, o requisito mínimo pode ser de até 2160 x 1440 de resolução.
