系統要求
------------------------------------------

Windows系統：
*Windows® 10, 11 或更高版本

Macintosh系統: 
*macOS® 10.15, 10.16 或更高版本

Windows和Macintosh的通用要求：
*需要64位操作系統
*2.0 GHz Intel Core i3或更高版本
*4 GB RAM或更高
*1.6 GB或更大的可用硬碟空間
*需要支持OpenGL 4.1的圖形卡（建議使用1920x1080）**
*需要連接互聯網啟動產品

**對於高點數密度顯示器，最低分辯率將根據操作系統建議的縮放級別而變化.例如，如果操作系統建議將縮放標準設定為200%，則最低要求可能高達2736x1824分辯率.或者，如果操作系統建議將縮放標準設定為150%，則最低要求可能高達2160x1440分辯率.
