Требования к системе
------------------------------------------


Windows: 
*Windows® 10, 11 или выше.


Macintosh: 
*macOS® 10.15, 10.16 или выше.


Общие требования для Windows и Macintosh:
*требуется 64-битная операционная система
*2.0 ГГц Intel Core i3 или выше
*4 ГБ оперативной памяти или больше
*1,6 ГБ свободного места на жестком диске или больше
*Требуется видеокарта с поддержкой 4.1 (рекомендуется 1920x1080)**
*Требуется подключение к сети Интернет для активации продукта




**Для дисплеев с высокой плотностью пикселей минимальное разрешение зависит от рекомендованного операционной системой уровня масштабирования. Например, если операционная система рекомендует уровень масштабирования 200%, минимальное разрешение может достигать 2736x1824. Или, если операционная система рекомендует уровень масштабирования 150%, минимальное требование может достигать разрешения 2160x1440.
