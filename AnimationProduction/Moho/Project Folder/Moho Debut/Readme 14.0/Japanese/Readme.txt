システム要件
------------------------------------------

Windows: 
*Windows® 10, 11 またはそれより新しいもの

Macintosh: 
*macOS® 10.15, 10.16 またはそれより新しいもの

Windows と Macintoshに共通する要件:
*64-bit OS 必須
*2.0 GHz Intel Core i3 またはそれ以上 
*4 GB RAM またはそれ以上
*1.6 GB のハードドライブ空き容量かそれ以上
*OpenGL 4.1 対応のグラフィックカードが必須 (1920x1080 推奨)**
*プロダクトアクティベーションにインタネット接続必須

**高画素密度ディスプレイでは、最小解像度はオペレーティングシステムで推奨されるスケーリングレベルによって異なります。例えば、OSが200％のスケーリングレベルを推奨する場合には、少なくとも2736x1824 の解像度が求められるでしょう。また、OSが150％のスケーリングレベルを推奨する場合には、2160x1440の解像度が求められます。
